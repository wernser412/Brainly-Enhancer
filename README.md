# Brainly Enhancer

**Última Actualización:** 21 de mayo de 2026

Este es un script Script de **Tampermonkey** para mejorar la experiencia en [Brainly](https://brainly.lat/) con herramientas visuales y productividad integradas.

## ✨ Características

### 🔍 Google Search Integrado
- Buscar texto seleccionado en Google sin salir de Brainly.
- Resultados en una ventana flotante.
- Ventana:
  - movible,
  - redimensionable,
  - y cerrable.

### 🖼️ Miniaturas Automáticas
- Muestra automáticamente las imágenes adjuntas de las tareas directamente en el feed.
- No necesitas abrir la publicación para ver el ejercicio.
- Las imágenes:
  - se precargan,
  - cargan rápido,
  - y se pueden abrir en tamaño completo.

### 🎨 Mejoras Visuales
- Interfaz más cómoda.
- Vista previa integrada.
- Transiciones suaves.

## 📥 Instalación
1. Instala la extensión [Tampermonkey](https://www.tampermonkey.net/) en tu navegador.
2. Haz clic en este enlace para instalar el script:  
   👉 **[Descargar Script](https://github.com/wernser412/Google-Enhancer/raw/refs/heads/main/Brainly-Enhancer.user.js)**
3. ¡Listo! Ahora puedes seleccionar texto en Brainly y buscarlo en Google.

## 🖥️ Uso

### Buscar en Google
1. Selecciona texto.
2. Haz clic en el icono de Google.
3. Se abrirá el cuadro flotante con resultados.

### Ver miniaturas
- Las tareas con adjuntos mostrarán automáticamente una miniatura.
- Haz clic en la imagen para abrirla en tamaño completo.
